# Azure Cloud File System

This project documents a cloud-based file storage workflow built using Microsoft Azure services.

## Overview
The system demonstrates how files can be uploaded, stored, and accessed securely in a cloud environment.

## Components
- Azure App Service
- Azure Blob Storage
- Shared Access Signature (SAS)

## Features
- File upload to cloud storage
- Secure access using SAS tokens
- Scalable cloud-based architecture

This project reflects my interest in building data-driven systems and scalable infrastructure that can support engineering and construction information workflows. It also demonstrates foundational understanding of system integration and automation in digital environments.
